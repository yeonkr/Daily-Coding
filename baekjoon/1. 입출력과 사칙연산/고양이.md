## 고양이

### 문제

아래 예제와 같이 고양이를 출력하시오.

<br/>

### 입력

없음.

<br/>

### 출력

고양이를 출력한다.

<br/>

### 예제 입력1

<br/>

### 예제 출력1

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

---

컴파일러가 문법과 헷갈릴만한 문자는 앞에 `\`를 붙여 문자열로 인식시켜야한다.

백 슬래시`\`를 사용하는 경우

- `\n` : 줄 바꿈 (Enter)
- `\\` : 백 슬래시
- `\'` : 작은 따옴표
- `\"` : 큰 따옴표

```js
console.log(
  `\\    /\\
 )  ( ')
(  /  )
 \\(__)|`
);
```
